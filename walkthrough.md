# AURA Assistant Setup & Operation Guide (Email Edition)

We have built a 100% free serverless personal assistant workflow that uses **Email notifications** instead of Telegram. Follow these step-by-step instructions to get everything configured and running in under 3 minutes.

---

## 🛠️ Step 1: Google Sheets Database Setup (1 Minute)

1. Open [Google Sheets](https://sheets.google.com) and create a **blank spreadsheet**.
2. Click **Extensions > Apps Script** in the top menu bar.
3. Replace the placeholder code with the contents of your [google_apps_script.js](file:///d:/Ritesh/antigravity/my-ai-agent/google_apps_script.js).
4. Click the **Save** disk icon, then click **Deploy > New Deployment**.
5. Configure the deployment details:
   * **Select Type**: Click the gear icon and select **Web App**.
   * **Description**: `Aura API`
   * **Execute As**: `Me`
   * **Who has access**: `Anyone` (This allows your dashboard to save logs securely).
6. Click **Deploy**. Authorize the permissions with your Google account.
7. **Copy the Web App URL** displayed in the success dialog (e.g., `https://script.google.com/macros/s/.../exec`).

---

## 🚀 Step 2: Deploying to GitHub (2 Minutes)

### 1. Push code to your Private GitHub Repository
1. Create a new **Private** repository on GitHub.
2. Push the files in this project workspace:
   * [index.html](file:///d:/Ritesh/antigravity/my-ai-agent/index.html)
   * [style.css](file:///d:/Ritesh/antigravity/my-ai-agent/style.css)
   * [script.js](file:///d:/Ritesh/antigravity/my-ai-agent/script.js)
   * [automation.py](file:///d:/Ritesh/antigravity/my-ai-agent/automation.py)
   * [.github/workflows/assistant.yml](file:///d:/Ritesh/antigravity/my-ai-agent/.github/workflows/assistant.yml)

### 2. Configure GitHub Pages (Free Hosting)
1. In your GitHub repository, go to **Settings > Pages**.
2. Under **Build and deployment > Source**, select **Deploy from a branch**.
3. Select your branch (usually `main` or `master`) and folder (`/root`), then click **Save**.
4. GitHub will host your dashboard at `https://<your-username>.github.io/<your-repo-name>/`.

### 3. Add secrets to GitHub Actions
1. In your GitHub repository, go to **Settings > Secrets and variables > Actions**.
2. Click **New repository secret** and add these variables:
   * `GOOGLE_SHEET_URL`: (The URL copied in Step 1)
   * `GEMINI_API_KEY`: (Your free Gemini API Key)
   * `ALERT_EMAIL`: (The email address where you want to receive alerts)

---

## ⚙️ Step 3: Connecting the Dashboard

1. Open your hosted GitHub Pages dashboard link (or open `index.html` locally in a browser).
2. Click **Settings** (gear icon in the bottom left corner).
3. Paste the following configuration values:
   * **Google Sheets Web App URL**: (From Step 1)
   * **Gemini API Key**: (Your key, which will let you chat with the AI chatbot directly in the dashboard)
   * **Alert Email Address**: (The email where you want to receive alerts)
4. Click **Save Configuration**. The indicator in the top right will turn green: **🟢 Synced with Sheet**.

---

## 💡 How to Use Your Dashboard

* **AI Chat Assistant**: Click the **AI Assistant** tab to brainstorm solutions, troubleshoot code, or get advice. It will use your Gemini API Key.
* **Prompt Architect**: Click **Prompt Architect** to generate highly optimized Prompts. Use the sliders/dropdowns to tailor details for Midjourney, DALL-E, Sora, or Runway.
* **Tasks & Alerts**: Add reminders. They will trigger in-browser alerts and send you an Email automatically at the configured time.
* **LinkedIn posts**: View your daily drafts generated by your GitHub Actions bot. Click **Copy Post** and open LinkedIn to publish with a single click.
